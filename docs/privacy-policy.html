Privacy Policy

TaxiVillage — это приложение для заказа такси, курьерских услуг, еды и межгородних поездок. Эта страница описывает, какие данные мы обрабатываем, зачем они нужны и как пользователь может запросить удаление аккаунта.

Какие данные мы собираем

  • номер телефона и данные аккаунта для входа в приложение;
  • геолокацию и адреса для такси, курьера и межгородних поездок;
  • данные профиля, включая имя и аватар, если пользователь их добавляет;
  • сообщения в чатах, историю заказов и бронирований;
  • фотографии и документы водителей и заведений;
  • push token для отправки уведомлений.

Зачем мы используем данные

  • для создания и выполнения заказов и бронирований;
  • для связи между пассажирами, водителями и заведениями;
  • для проверки документов водителей и заведений;
  • для отправки уведомлений о статусах поездок, заказов и приглашений;
  • для безопасности аккаунта и предотвращения злоупотреблений.

Передача данных третьим лицам

TaxiVillage может использовать сторонние сервисы для инфраструктуры приложения, базы данных, хранения файлов, карт и push-уведомлений. Эти сервисы используются только для работы приложения.

Хранение и удаление данных

Пользователь может удалить аккаунт внутри приложения. После запроса на удаление аккаунта аккаунт помечается как удаленный, push token очищается, а персональные поля профиля заменяются на обезличенные значения.

Если пользователь не может зайти в приложение, он может использовать внешнюю страницу удаления аккаунта: Delete Account

Контакты

По вопросам приватности и удаления аккаунта можно написать на: support@taxivillage.app
